Ubuntu 22.04에서 Desktop 모드로 부팅 끄고 켜기

18 June 2022

우분투 22.04 데스트탑 버전을 설치하면 자동으로 데스크탑으로 부팅된다.
서버 버전을 설치한 것과 같이 데스크탑없이 부팅하여 사용하고자 할 경우에는 systemctl 명령으로 간단히 GUI 모드를 켜고 끌 수 있다.

GUI로 부팅 끄기

systemctl 명령으로 GUI 모드를 끄고 재부팅하면 된다.

$ sudo systemctl set-default multi-user
Created symlink /etc/systemd/system/default.target → /lib/systemd/system/multi-user.target.
$ reboot

부팅되면 아래 이미지 처럼 터미널이 표시된다.

GUI로 부팅 켜기

systemctl 명령으로 GUI 모드를 켜고 재부팅하면 된다.

$ sudo systemctl set-default graphical
Removed /etc/systemd/system/default.target.
Created symlink /etc/systemd/system/default.target → /lib/systemd/system/graphical.target.
$ reboot
