개인 취미로 운영하는 사이트입니다.
항상 정확한 정보를 전달하고자 최선을 다하고 있지만,
간혹 부정확한 정보가 전달될 수 있으니 실무에 적용시에는 항상 본인이 직접 확인하여 사용하시기 바랍니다.

사용된 오픈소스

  • JBake를 이용하여 정적 html을 만들고 github.com에 커밋하여 홈페이지를 운영합니다.
  • Bootstrap Bootstrap v5.0.0-best3를 사용합니다.
  • jQuery jQuery 3.6.0을 사용합니다.
  • JavaScript code prettifier를 이용하여 코드를 표시합니다.
